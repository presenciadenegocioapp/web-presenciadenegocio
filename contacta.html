Condiciones Generales

Presencia de Negocio es un sitio web informativo destinado a mostrar servicios de creación de páginas web para negocios.

La información publicada en esta web tiene carácter informativo y no constituye por sí sola una contratación directa del servicio.

La contratación, condiciones económicas, forma de pago y demás condiciones se informarán al interesado antes de formalizar cualquier servicio.

  • La contratación no se realiza automáticamente desde esta web.
  • Los servicios se ofrecen tras contacto previo con el interesado.
  • Los pagos podrán gestionarse a través de plataformas externas como Hotmart.
  • Las condiciones de cancelación y reembolso se comunicarán antes de la contratación.
  • Está prohibida la copia, reproducción o uso del contenido de esta web sin autorización expresa.
  • El usuario se compromete a hacer un uso adecuado del sitio web conforme a la ley y la buena fe.
